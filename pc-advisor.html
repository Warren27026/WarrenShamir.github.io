PC-ADVISOR

Une application web Angular pour guider les utilisateurs dans le choix d'un ordinateur adapté à leurs besoins, avec des parcours éducatifs et personnalisés.

Détails

PC-ADVISOR est une application développée avec Angular qui propose trois parcours (Débutants, Connaisseurs, Éducatif) pour aider les utilisateurs à choisir un ordinateur en fonction de leurs besoins (CPU, GPU, RAM, stockage, budget). Le projet inclut un système de recommandation basé sur des filtres dynamiques, une interface utilisateur responsive, et des explications éducatives. Les données sont gérées via des fichiers JSON, avec une logique de filtrage optimisée pour des recommandations pertinentes.

Technologies utilisées

Angular TypeScript HTML CSS JSON

Durée

Mai-Juin 2025

Résultats et Impact

L'application PC-ADVISOR offre une expérience utilisateur intuitive, avec des recommandations précises basées sur les préférences des utilisateurs. Le parcours éducatif a permis d'améliorer la compréhension des composants informatiques, tandis que les parcours Débutants et Connaisseurs ont simplifié le choix pour des publics variés. Ce projet a renforcé mes compétences en développement web, design d'interface, et gestion de données, avec un potentiel d'extension pour inclure des comparaisons en temps réel ou une base de données plus large.

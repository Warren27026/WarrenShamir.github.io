PC-ADVISOR

Application web Angular qui guide l’utilisateur vers un ordinateur adapté à ses besoins grâce à un parcours personnalisé et éducatif.

Durée

Type

Application Web

Description

PC-ADVISOR est une application Angular qui propose 3 parcours (Débutants, Connaisseurs, Éducatif) pour aider à choisir un ordinateur selon l’usage (bureautique, gaming, montage, etc.) et des contraintes (budget, CPU, GPU, RAM, stockage).

  • Recommandation via filtres dynamiques
  • Parcours éducatif pour comprendre les composants
  • Interface responsive
  • Données gérées via fichiers JSON

Technologies

Angular TypeScript HTML CSS JSON
Voir le projet

Résultats & Impact

PC-ADVISOR propose une expérience utilisateur fluide et des recommandations adaptées à différents profils. Le parcours éducatif améliore la compréhension des composants (CPU, GPU, RAM...), tandis que les parcours Débutants et Connaisseurs accélèrent la prise de décision. Ce projet a renforcé mes compétences en Angular, en conception d’UI, et en gestion de données structurées (JSON + logique de filtrage).

PC-ADVISOR – Recommandation d’ordinateur

Application web Angular permettant de recommander un ordinateur adapté aux besoins de l’utilisateur (bureautique, gaming, montage, etc.) via des parcours personnalisés et un module éducatif sur les composants.

Angular TypeScript JSON Filtres dynamiques Responsive

Période

Mai – Juin 2025

Type

Description

PC-ADVISOR propose trois parcours : Débutants, Connaisseurs et Éducatif. L’utilisateur indique ses besoins (usage, budget, performances) et l’application fournit une recommandation adaptée.

  • Filtrage intelligent sur CPU, GPU, RAM, stockage et budget
  • Parcours éducatif expliquant les composants informatiques
  • Interface responsive pensée pour desktop et mobile
  • Données structurées via fichiers JSON

Technologies utilisées

Angular TypeScript HTML CSS JSON
